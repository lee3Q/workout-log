# 건강식품 판단 프레임워크 v1.0
> 2026-04-13 교차검증(Opus + Sonnet) 완료

## 0단계: 현 상황 진단 + 의지 확인

식단/운동/수면에서 개선할 여지가 있는가?
- Yes → 개선할 의지가 있는가?
  - Yes → **먼저 그걸 개선하라. 건강식품은 그 후에.**
  - No → 차선책으로 건강식품 진입. 1단계로.
- No (물리적 한계: 소화기 질환, 흡수 장애 등) → 보상 전략으로 건강식품 진입. 1단계로.

## 1단계: 안전한가?

- NIH LiverTox에서 간 독성 보고 확인
- Drugs.com interaction checker로 약물 상호작용 확인
- 개인 맥락 필터: 기저질환, 복용 약물, 임신/수유, 나이
- **해가 있으면 여기서 탈락. 더 볼 필요 없음**

## 2단계: 누가 파는가?

- MLM/다단계 유통 구조인가?
- 과대광고 채널(인플루언서, 의사 마케팅)인가?
- 제조사 GMP 인증 여부
- 3자 품질 검증: NSF, USP, ConsumerLab 인증
- **유통구조가 왜곡되면 이후 단계의 정보 자체가 오염됨**

## 3단계: 나에게 필요한가?

- 결핍 교정형 (비타민D, 철분, B12 등): 혈액검사로 확인. 기준값은 보수적 기준 채택
- 기능 조절형 (오메가3, 프로바이오틱스 등): 목표 지표를 명확히 설정
- 0단계 진단 결과에 따라 판단 기준이 달라짐:
  - 물리적 한계 → 결핍 보상이 확실하면 강하게 정당화됨
  - 의지력 차선책 → 필요성은 인정하되 효과 근거(4단계) 기준은 낮추지 않음

## 4단계: 효과 근거가 있는가?

- Examine.com 근거 등급 확인 (일반인 실행 가능)
- **반증 의무: "효과 없다는 리뷰도 하나 찾아라"**
- 단기 효과(수면, 에너지) vs 장기 예방(심혈관, 암) 구분 — 근거 종류가 다름
- NNT(Number Needed to Treat) 확인 가능하면 확인
- 빨간 깃발: 숫자 없는 "~에 좋다", 동물실험만, 결핍 환자 연구를 정상인에게 일반화

## 5단계: 이 제품의 용량/형태가 맞는가?

- 연구에서 효과 나온 용량 vs 제품 용량 비교
- 흡수 가능한 형태인지 (예: 마그네슘 glycinate vs oxide)
- 추출물 내 실제 활성 성분 함량

## 6단계: 비용 대비 가치가 있는가?

- 음식으로 대체 가능한가?
- 대체 가능하더라도 **실제로 그만큼 먹을 수 있는가/먹을 것인가?**
- 월 비용 vs 식단 개선 비용

## 메타 규칙 (모든 단계 관통)

> "나는 이미 이 제품을 사고 싶다고 결정했는가? 그렇다면 0단계부터 다시."
