# 단백질 함량표 (100g당, 생고기 기준)

> 출처: USDA FoodData Central (fdc.nal.usda.gov), 식약처 식품영양성분DB

## 고기류

| 식품 | 단백질/100g | 비고 |
|------|:-----------:|------|
| 삼겹살 | ~10g | 지방 53%, 단백질 효율 낮음 |
| 목살 | ~17g | 삼겹살보다 단백질 1.7배 |
| 닭가슴살 | ~23g | 최고 효율 |
| 소고기 등심/안심 | ~20-21g | 고품질 단백질 |

## 기타 단백질 식품

| 식품 | 단백질 | 비고 |
|------|:------:|------|
| 달걀 1개 (50g) | ~6.3g | 완전 단백질 |
| 우유 200ml | ~6.5g | 칼슘 보충 겸용 |
| 두부 100g | ~8.5g | 식물성 |
| 참치캔 100g | ~24-25g | 닭가슴살급 |
| 프로틴 쉐이크 1회 | ~25-30g | 보충제 |
| 그릭 요거트 100g | ~10g | 간식으로 우수 |

## 실전 계산 예시

| 식사 | 단백질 |
|------|:------:|
| 삼겹살 600g (1근) | ~60g |
| 목살 600g (1근) | ~102g |
| 닭가슴살 200g | ~46g |
| 달걀 3개 + 우유 200ml | ~25g |
| 참치캔 1캔 (150g) | ~37g |

## 목표 달성 팁
- 일일 목표: 114g+ (체중 71kg x 1.6g/kg)
- 삼겹살보다 목살이 단백질 1.7배 → 같은 양 먹어도 차이 큼
- 간편 보충: 달걀 2개(12g) + 우유(6.5g) + 그릭요거트(10g) = 28g 추가
