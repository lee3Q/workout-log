## 1. 스쿼트 (스미스 머신)
**목표: 3세트 x 10회 | 무게: 오늘 찾기 (빈 바 → 20kg → 30kg 올려보기)**

### 자세
1. 바를 승모근 위(목 아래 뼈 바로 밑)에 걸친다. 목에 직접 닿으면 안 됨.
2. 발은 어깨 너비, 발끝 살짝 바깥(30도)으로.
3. 숨 크게 들이쉬고 배에 힘 → 엉덩이를 뒤로 빼면서 내려간다.
4. **허벅지가 바닥과 평행**될 때까지 내려간다 (최소 목표).
5. 발 뒤꿈치로 밀면서 올라온다. 무릎이 발끝 안쪽으로 모이면 안 됨.

### 체크포인트
- 허리 둥글게 말리면 ❌ → 가슴 펴고 시선 정면
- 무릎이 발끝보다 앞으로 나와도 OK (자연스러운 범위 내)
- 내려갈 때 3초, 올라올 때 2초

### 대체: 고블릿 스쿼트
- 덤벨 하나를 가슴 앞에 세로로 들고 같은 동작

## 2. 레그프레스
**목표: 3세트 x 12회 | 무게: 오늘 찾기 (가벼운 것부터)**

### 자세
1. 등과 엉덩이를 시트에 완전히 밀착.
2. 발을 발판 중앙~약간 위에, 어깨 너비로 놓는다.
3. 안전 잠금 해제 → 천천히 무릎을 구부려 내린다.
4. 무릎이 90도가 될 때까지 내린 뒤 밀어 올린다.
5. **다리 완전히 펴지 마** (무릎 잠금 ❌). 살짝 굽힌 상태에서 멈춤.

### 체크포인트
- 엉덩이가 시트에서 뜨면 ❌ → 너무 깊이 내린 거. 범위 줄이기.
- 무릎이 안쪽으로 모이면 ❌ → 발끝 방향으로 밀기.
- 호흡: 내릴 때 들이쉬고, 밀 때 내쉬고.

### 대체: 불가리안 스플릿 스쿼트
- 뒤쪽 발을 벤치 위에 올리고, 앞다리로 스쿼트. 덤벨 양손에 들기.

## 3. 레그컬 (라잉/시티드)
**목표: 3세트 x 12회 | 무게: 오늘 찾기**

### 자세 (라잉 레그컬)
1. 엎드려서 패드를 발목 뒤(아킬레스건 위)에 건다.
2. 무릎 위치가 머신 축과 일치하도록 조절.
3. 허벅지 뒤쪽(햄스트링)에 힘 주면서 발을 엉덩이 쪽으로 당긴다.
4. 최대로 당긴 후 1초 멈추고, 천천히 내린다.

### 체크포인트
- 엉덩이가 들리면 ❌ → 무게 낮추기.
- 종아리에만 힘이 가면 → 발끝을 몸 쪽으로 당기기(도시플렉션).

### 대체: 루마니안 데드리프트 (덤벨)
- 덤벨 양손, 무릎 살짝 굽힌 채 → 엉덩이를 뒤로 빼며 상체 숙이기 → 햄스트링 당기는 느낌에서 멈추고 올라오기.

## 4. 숄더프레스 (머신)
**목표: 3세트 x 10회 | 무게: 오늘 찾기**

### 자세
1. 시트 높이 조절 — 손잡이가 어깨 높이에 오도록.
2. 등을 시트에 완전 밀착. 허리 아치 과도하게 ❌.
3. 손잡이를 잡고 위로 밀어 올린다. 팔꿈치 완전히 펴지 않게.
4. 천천히 내려서 시작 위치로.

### 체크포인트
- 어깨가 귀 쪽으로 올라가면 ❌ → 어깨 내리고 가슴 펴기.
- 목에 힘 들어가면 → 무게 낮추기.

### 대체: 덤벨 숄더프레스
- 벤치를 80-90도로 세우고 앉아서, 덤벨을 귀 옆에서 위로 밀기.

## 5. 행잉 레그레이즈
**목표: 3세트 x 할 수 있는 만큼 (10-15회)**

### 자세
1. 풀업바에 매달린다 (팔 완전히 펴고, 어깨너비 그립).
2. 몸 흔들림 잡고 → 다리를 모아서 앞으로 올린다.
3. **최소 90도**(허벅지가 바닥과 평행)까지 올리기.
4. 1초 멈추고 천천히 내린다. 그냥 떨어뜨리면 ❌.

### 체크포인트
- 몸이 흔들리면 → 올리기 전에 멈추고, 반동 없이.
- 다리 쭉 펴기 힘들면 → **무릎 구부려서 올려도 OK** (니레이즈).
- 악력이 모자라면 → 스트랩 사용 or 벤치에서 라잉 레그레이즈로 대체.

### 대체: 라잉 레그레이즈
- 벤치에 눕고 → 다리 쭉 펴서 90도까지 올리고 → 천천히 내리기. 허리가 뜨면 ❌.

- 템포: 올릴 때 2초, 내릴 때 3초
