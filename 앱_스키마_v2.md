## v2 데이터 모델

### sessions
```js
{
  id: "2026-04-08",        // ISO 날짜 = id
  date: "2026-04-08",
  weekday: 3,              // 0=일, 3=수
  split: "wed_pump" | "thu_push_squat" | "fri_pull_deadlift" | "free",
  exercises: [ExerciseLog],
  duration_min: number | null,
  completed: boolean
}
```

### ExerciseLog
```js
{
  exercise_id: "back_squat",
  order: 1,
  sets: [
    { set_number: 1, weight_kg: 50, reps: 5, completed: true },
    { set_number: 2, weight_kg: 50, reps: 5, completed: true },
    ...
  ]
}
```

### exercises (정의 — 코드 상수)
```js
{
  id: "back_squat",
  name: "백 스쿼트",
  category: "compound",        // compound | accessory
  is_main_lift: true,          // 5x5 진행 룰 적용 대상
  default_sets: 5,
  default_reps_min: 5,
  default_reps_max: 5,
  progression_rule: "linear_5x5",
  increment_kg: 2.5
}
```

### progression_state (운동별 진행 상태 — 별도 키)
```js
{
  exercise_id: "back_squat",
  current_target_weight_kg: 50,    // 다음 세션 목표
  consecutive_failures: 0,
  last_success_date: "2026-04-08",
  last_failure_date: null,
  weeks_at_current_weight: 1
}
```

### localStorage 키 구조
```
workoutState_v2 = {
  sessions: { "2026-04-08": Session, ... },
  progression: { "back_squat": ProgressionState, ... },
  meta: { last_session_date: "...", version: 2 }
}
```

마이그레이션: 기존 `workoutState` 키는 무시하고 새 키 `workoutState_v2` 사용. 사용자가 v1 데이터 필요 없다고 함.

## 핵심 기능 (v2)

### 1. 오늘 분할 자동 결정
- 요일 → split 자동 매핑
- 자유 요일(토일월화)이면 "자유 영역" 화면:
  - "오늘은 자유 — 휴식, 자전거, 맨몸 중 선택"
  - 세트 입력 X. 단순 메모만 (선택적).

### 2. 운동 카드 — "이전 vs 오늘"
각 운동 카드 상단에:
```
[백 스쿼트]
지난번 (4/2): 47.5kg × 5/5/5/5/5 ✓
오늘 목표: 50kg × 5×5
```

세트 입력 필드는 기본값으로 목표 무게 자동 채움.

### 3. 자동 진행 판정 (세션 종료 시)
각 메인 컴파운드별:
```js
function judge(exercise_id, sets) {
  const all_5_complete = sets.every(s => s.completed && s.reps >= 5);
  const state = progression[exercise_id];

  if (all_5_complete) {
    state.current_target_weight_kg += state.increment_kg;
    state.consecutive_failures = 0;
    state.last_success_date = today;
  } else {
    state.consecutive_failures++;
    state.last_failure_date = today;

    if (state.consecutive_failures >= 2) {
      // 디로드 권장 알림
      state.current_target_weight_kg *= 0.9;  // -10%
      state.consecutive_failures = 0;
      notify("디로드 권장 — " + exercise_id + " -10%");
    }
  }
}
```

### 4. 정체 감지
- `weeks_at_current_weight ≥ 4` → "디로드 검토" 알림
- 사용자가 디로드 수락 시 -10% 적용

### 5. 히스토리 (간단)
- 메인 컴파운드별 무게 진행 그래프 (선 그래프, vanilla canvas 또는 단순 표)
- 1RM 추정 (Epley 공식: `weight × (1 + reps/30)`)

### 6. 결과 복사 (v1 유지)
세션 종료 시 결과 텍스트 복사 → /운동 세션에 붙여넣기.
형식 예시:
```
2026-04-09 (목, Push+Squat)
- 백 스쿼트: 50kg × 5/5/5/5/5 ✓
- 벤치프레스: 40kg × 5/5/5/5/4 (마지막 1회 미달)
- 머신 OHP: 25kg × 8/7/7/6
...
```
