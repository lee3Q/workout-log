# Health - 운동 기록 & 건강 관리

이 폴더는 운동 기록, 프로그램 맥락, 참고 자료를 관리하는 곳이다.

## 디렉토리 구조
- `CONTEXT.md` — 프로그램 맥락 (무게 기준점, 스케줄, 주차별 메모). 세션 시작 시 반드시 먼저 읽을 것.
- `YYYY-MM-DD.md` — 일별 운동 기록
- `YYYY-MM-DD-가이드.md` — 운동 가이드 (자세/무게 팁)
- `단백질함량표.md` — 단백질 섭취 계산 기준표
- `치트시트.md` — 운동 치트시트

## 운동 세션
- `/운동` 스킬로 운동 보고 세션 실행
- 기록 저장 후 반드시 git add → commit → push
